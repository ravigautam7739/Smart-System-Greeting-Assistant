# 🤖 Smart System Greeting Assistant

A Python-based **voice greeting assistant** that welcomes the user with the current time, battery status, and a random motivational/system message.

This project creates a **Jarvis-style startup greeting experience** using text-to-speech automation.

---

# 🚀 Features

✔ Voice-based greeting system 🔊
✔ Announces current time ⏰
✔ Reports battery percentage 🔋
✔ Random motivational/system messages
✔ Lightweight and beginner-friendly

---

# 🛠 Technologies Used

* Python
* Pyttsx3 (Text-to-Speech)
* Datetime
* Psutil
* Random module

# 🧠 How It Works

1. Program fetches:

   * Current time
   * Battery percentage

2. Selects a random message from predefined greetings

3. Combines all information into a voice greeting

4. Uses text-to-speech to speak the message aloud
